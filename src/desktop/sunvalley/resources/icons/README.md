# ZirconOS Sun Valley Icons

WinUI 3 / Sun Valley style icon set for ZirconOS.
Design characteristics:
- Rounded geometry matching 8px system corner radius
- Consistent 32x32 viewport with 2px padding
- Light/dark adaptive color tokens
- Gradient fills for depth perception
- Crystal-blue accent (#4CB0E8) with warm highlight (#F0A030)
- Slightly bolder strokes compared to Fluent (0.6-0.7px vs 0.5px)
