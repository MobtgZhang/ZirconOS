# ZirconOS Sun Valley Cursors

Cursor set for ZirconOS Sun Valley theme.
Design follows WinUI 3 cursor guidelines:
- Clean monochrome design with crystal-blue accent hints
- Slightly thicker stroke than Fluent (1.2px vs 1px body outline)
- Rounded tips and corners matching 8px system radius
- 14x20 base size, DPI-scalable
- Shadow uses deeper navy (#0A1628) for Mica integration

## Cursor Inventory
