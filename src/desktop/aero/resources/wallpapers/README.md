## Design Guidelines

- **Color palette:** ZirconOS blue tones (#0A3A6A - #A8D8F0)
- **Accent colors:** Teal (#0ABDC6, #4DEEEA), warm orange (#F7A41D) as Zig language nod
- **Style:** Clean, modern, abstract with glass/crystal motifs
- **Resolution:** 1920x1080 minimum, SVG format for scalability
- **Format:** SVG preferred for resolution independence; raster exports at 4K available
- All designs are original to the ZirconOS project
