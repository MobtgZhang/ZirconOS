# ZirconOS Luna Wallpapers

Original wallpaper designs for ZirconOS Luna (Windows XP style) theme.

## Included Wallpapers

| File | Resolution | Theme | Description |
|------|-----------|-------|-------------|
| `bliss.svg` | 1024x768 | Blue (Default) | Rolling green hills under vivid blue sky with white clouds |
| `bliss_olive.svg` | 1024x768 | Olive Green | Autumn-toned rolling hills with warm amber sky |
| `bliss_silver.svg` | 1024x768 | Silver | Cool grey-toned landscape with overcast silver sky |

## Design Guidelines

- **Color palette:** Vivid blues, greens matching the Luna Blue scheme
- **Style:** Natural landscape with rolling hills, inspired by Bliss
- **Resolution:** 1024x768 base, SVG format for scalability
- **Format:** SVG preferred for resolution independence
- All designs are original to the ZirconOS project
