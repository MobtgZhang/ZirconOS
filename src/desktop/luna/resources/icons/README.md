## Design Language

All icons follow the Windows XP Luna visual language:

- **3D Shading**: Bright highlights on top-left, deep shadows on bottom-right
- **Vivid Colors**: Saturated blues, greens, yellows, and oranges
- **Orange Highlights**: Distinctive orange accent details throughout
- **Warm Tones**: Beige/cream secondary surfaces (ECE9D8)
- **Drop Shadows**: Subtle ground shadows for depth

## Technical Details

- Format: SVG (Scalable Vector Graphics)
- ViewBox: `0 0 32 32` for all icons
- All icons use `<defs>` for gradient and filter definitions
- No external dependencies or linked resources
- Compatible with standard SVG renderers
